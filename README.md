#  My Malloc

A custom implementation of the `malloc` memory allocation function in C.  
This project was built to better understand how **memory management** works under the hood.

> ⚠️ This project is for learning purposes only, not intended for production use.

---

## 📚 About

This project reimplements the standard `malloc` function from scratch in C,  
exploring concepts like :
- Heap management
- Memory alignment
- Block allocation and fragmentation

---

## 🛠️ Compilation

Clone the repository and run `make` at the **root of the project** :

```bash
git clone https://github.com/nathch-17/memory-allocator.git
cd memory-allocator
make
````
